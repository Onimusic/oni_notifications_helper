oni_notifications_helper.onitificator_telegram.send_document(bot_token: str, chat_id: str, document_url: str, caption: str = '') → Response¶

Envia um arquivo qualquer para um chat no Telegram. :param bot_token: Token recebido pelo bot otfather no Telegram. :param chat_id: Id do usuário ou do chat. No caso dos chats, começa com @ (envie com o @ nesse caso). :param document_url: Url da imagem ou id de um InputFile existente nos servidores do Telegram. Não encode o texto. :param caption: Legenda a ser enviada junto com o arquivo, na mesma mensagem. Não encode o texto. Isso será feito aqui :param mesmo. Limite de 1024 caracteres após encoding.:

Returns: response da requisição feita ao Telegram.
